CodeTurret is an automated security architect designed for the era of vibe-coding. It integrated with your repository to turn vibe-coded slop ready for production through various security algorithms, dual-pass AI architecture, and automatically fixes them for you by posting a PR

---

## Key Features
* GitHub Integration - integrates with your repository to make PRs for fixed code
* Security Algorithms - algorithms built to scan and find possbile vulnerable snippets, and flags it
* Dual-Pass AI Architecture - rapid triage via Gemini Flash models, and a deep-tissue analysis with Gemini Pro
* Ask Cortex - A natural language security consultant powered by Snowflake for you to query details like who commited what
* Semantic Search - Finds vulnerabilities to specific CVEs using vector embeddings
* Real-time Logs - live logs & security heatmaps in a minimalist UI
